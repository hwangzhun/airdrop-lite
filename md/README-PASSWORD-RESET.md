# 忘记管理员密码？重置方法

如果忘记了管理员密码，有以下几种方法可以重置：

## 方法一：删除配置文件（最简单） ⭐

这是最简单的方法，删除配置文件后系统会自动恢复默认密码（admin.json）

---

## 方法二：使用重置脚本（推荐） 🔧

使用项目提供的重置脚本，可以重置为默认密码或自定义密码。

### 重置为默认密码（admin123）：

```bash
node scripts/reset-password.cjs
```

### 重置为自定义密码：

```bash
node scripts/reset-password.cjs 你的新密码
```
